# WordMetrics

A professional, one-page word counter tool built with **Next.js**, **TypeScript**, and **Tailwind CSS**.

## Features

- **Real-time Stats**: Instantly see word, character, and line counts as you type.
- **Glassmorphism Design**: Sleek, modern UI with a premium feel.
- **Dark/Light Mode**: Responsive theme toggle with system preference detection.
- **Mobile Friendly**: Fully responsive layout that looks great on all devices.
- **Privacy Focused**: All calculations happen client-side in your browser.
- **Quick Actions**: One-click Copy and Clear functionality.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Open in Browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
